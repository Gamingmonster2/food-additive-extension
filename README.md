فاحص الغذاء الآمن: هو إضافة ذكية لمتصفح جوجل كروم مصممة لحماية صحة المستهلك. تتيح الإضافة للمستخدمين فحص المواد المضافة للأغذية (E-Numbers) إما عبر كتابة الرمز أو باستخدام كاميرا الهاتف كـ ماسح ضوئي سريع. يقوم النظام بتحليل المادة فوراً وإظهار مستوى خطورتها (آمن، تنبيه، خطر) مع شرح للأضرار المحتملة وحالتها القانونية دولياً، مما يجعل التسوق الصحي أسهل وأسرع.
